# AI Tool Usage Statement

This project made limited and transparent use of AI-based tools as a learning and productivity aid, in accordance with the course AI usage policy.

## Tools Used
- ChatGPT (OpenAI)

## Nature of Use
AI tools were used for:
- Debugging assistance and interpretation of error messages,
- Clarifying the usage of Python libraries (e.g., pandas, scikit-learn, matplotlib),
- Reviewing and improving code structure and readability,
- Helping draft documentation sections (README, methodology descriptions).

## What AI Was Not Used For
- AI tools did not generate the full project code,
- AI tools were not used to design the research question,
- All modeling choices, data processing steps, and result interpretations were implemented and validated by the author.

All final code, analyses, and written content were reviewed, understood, and modified by the author before submission.
